# 🌍 TripWay

O **TripWay** é um guia de viagens moderno e responsivo, desenvolvido para ajudar usuários a descobrirem novos destinos, planejarem suas viagens e viverem experiências inesquecíveis.

---

## ✈️ Sobre o projeto

O TripWay foi criado com o objetivo de facilitar o planejamento de viagens, reunindo informações sobre destinos, pacotes e serviços em uma interface simples, organizada e acessível.

A plataforma permite que o usuário explore diferentes opções de viagem de forma intuitiva, tornando todo o processo mais prático e inspirador.

---

## 🚀 Funcionalidades

- 📍 Exploração de destinos
- 📝 Formulário de planejamento de viagem
- 📱 Layout totalmente responsivo
- 🎯 Interface simples e intuitiva
- 🌐 Organização clara das informações

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em responsividade, adaptando-se a diferentes tamanhos de tela:

- 📲 Mobile  
- 💻 Tablet (≥ 768px)  
- 🖥️ Desktop (≥ 1024px)  

---

## 🎨 Layout

O design é baseado em cartões (cards), proporcionando uma visualização limpa e moderna das informações.
